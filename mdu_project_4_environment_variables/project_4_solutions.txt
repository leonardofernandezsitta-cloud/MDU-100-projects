Mathematical Thinking Exercise (Project #4)

Problem: A system has 1,000 users. Each user has a unique environment variable called USER_SECRET with a random 4-digit PIN (0000 to 9999).

If an attacker can run ps auxe on the system, what is the probability they can see at least one user's PIN?

Step 1: Total possible PINs = 10,000 (0000 to 9999)

Step 2: Probability a specific user's PIN is guessed randomly = 1/10,000 = 0.0001

Step 3: But the attacker is not guessing — they are reading the environment variable directly with ps auxe. If they can see it, they know it with 100% certainty.

Step 4: So the question becomes: What is the probability that at least one user has a process running with their USER_SECRET exposed?

Assume: At any given time, 10% of users have a process running (100 users). Each running process exposes its environment.

Answer: If 100 users have running processes, the attacker sees 100 PINs directly. Probability is 100% (they will see them all).

Moral of the story: Environment variables are not a secret store. Anyone who can run ps auxe on the same machine can read them.
