Math Problem 1 (From Challenge 1 - Environment Variables)

Problem: Environment variable names in Linux can contain:

    Uppercase letters (A-Z): 26 possibilities

    Lowercase letters (a-z): 26 possibilities

    Underscore (_): 1 possibility

    Digits (0-9): 10 possibilities (but cannot be the first character)

Questions:

    How many possible 2-character environment variable names are there?

        First character: A-Z, a-z, or _

        Second character: A-Z, a-z, _, or 0-9
        
Answer:

first_character = a = 26(a-z) + 27(A-Z+_) = 53
second_character = b = 26(a-z) + 37(A-Z+_+0-9) = 63

probability for first character = a = 53
probability for second character = b = 63
possible 2 character names = a * b = 53 * 63 = 3,339

    How many possible 3-character environment variable names are there?

        First character: 53 possibilities (A-Z, a-z, _)

        Second character: 63 possibilities (A-Z, a-z, _, 0-9)

        Third character: 63 possibilities (same as second)
        
Answer:
        
probability for first character = a = 53
probability for second character = b = 63
probability for second character = c = 63
possible 2 character names = (a * b) * c = (53 * 63) * 63 = 210,357

Math Problem 2 (From Challenge 2 - File Permissions)

Problem: File permissions in Linux are three octal digits (0-7). Each digit is the sum of:

    4 = read (r)

    2 = write (w)

    1 = execute (x)

Questions:

    Convert 640 to rwx string. Show your work. 
    
Answer: 

owner = 4 + 2 = rw-
group = 4 = r--
others = 0 = ---
rwx = rw-r-----

    Convert 755 to rwx string. Show your work.
    
Answer: 

owner = 4 + 2 + 1 = rwx
group = 4 + 1 = r-x
others = 4 + 1 = r-x
rwx = rwxr-xr-x


    Convert rwxr-xr-- to octal. Show your work.
    
rwx = 4 + 2 + 1 = 7
r-x = 4 + 1 = 5
r-- = 4
octal = 754

Math Problem 3 (From Challenge 3 - Probability)

Problem: A file permission has three octal digits (owner, group, others). Each digit ranges from 0 to 7 (8 possibilities).

Questions:

    How many total possible permission combinations exist? Show your calculation.
    
owner = a = 8
group = b = 8
others = c = 8
possible permission combinations = (8*8) * 8 = 64 * 8 = 512

    If a permission is set completely at random, what is the probability that the owner digit is exactly 6?
    Express this probability as a fraction, decimal, and percentage.
    
1/8 because owner has 8 possible numbers 0-7 and can be only 1 = 1/8 = 0.125 = 12.5%
